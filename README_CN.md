# **Easy-Compact**
  Easy-Compact是一个将Windows系统自带的Compact命令集成到右键菜单中的注册表项。通过它，用户可以方便地对文件或文件夹直接进行Compact压缩操作，无需手动输入命令。
  
  Compact压缩能够在Windows系统下有效减少文件占用的磁盘空间，同时不影响文件的正常读取和使用，是一种高效、无损的系统级压缩方式。
通过本注册表项，用户可选择XPRESS4K、XPRESS8K、XPRESS16K、LZX四种不同算法对文件或文件夹进行压缩，并支持对应的解压缩操作。
